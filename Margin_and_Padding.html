Margin and Padding

Padding (Inside the border)

For adding gaps inside the border between borders and text, we will use padding
syntax: padding : 16px;
The above syntax will put gaps of 16px on all sides of this paragraph.

Different paddings on different size

padding: 100px;
(The below four lines will override the above one)
If one of the values missing from the below, that side will automatically hold the value of padding for that corresponding side.
padding-top:10px;
padding-right:10px:
padding-bottom: 10px;
padding-left: 10px;

Shorthand version of padding

padding : top right bottom left;

padding : top-bottom left-right

Margin (Outside the border)

For adding gaps outside the border , we will have "margin"
syntax: margin:30px;
The above syntax will put the gap between outside the border

Everything is same as padding
